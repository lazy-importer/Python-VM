# Resources

## Official documentation

- [Python `dis` module](https://docs.python.org/3/library/dis.html): inspect CPython bytecode.
- [Python `marshal` module](https://docs.python.org/3/library/marshal.html): internal binary serialization used here for metadata.
- [Python `struct` module](https://docs.python.org/3/library/struct.html): pack and unpack binary values.
- [CPython interpreter internals](https://github.com/python/cpython/blob/main/InternalDocs/interpreter.md): official internal notes about the bytecode interpreter.
- [CPython source code](https://github.com/python/cpython): reference implementation of Python.

## Related repositories

- [Byterun](https://github.com/nedbat/byterun): a Python implementation of a Python bytecode execution VM.
- [`bytecode`](https://github.com/MatthieuDartiailh/bytecode): a Python library to generate and modify Python bytecode.
- [python-control-flow](https://github.com/rocky/python-control-flow): tools related to control-flow information from Python bytecode.
